poppler.cpp package¶

The poppler.cpp package is the direct binding to the poppler-cpp library. You usually do not need to interract directly with it, as all objects are wrapped using native Python objects in the poppler module.

This module mostly follows the C++ code. Therefore, you can refer to the poppler-cpp documentation.

Module contents¶
